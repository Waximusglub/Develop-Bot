# Bot de Discord

Este es un bot de Discord desarrollado en JavaScript utilizando la librería Discord.js.

## Descripción

El bot de Discord está diseñado para proporcionar funcionalidades útiles y entretenidas a tu servidor de Discord. Puede ser personalizado y ampliado según las necesidades específicas de tu comunidad.

## Funcionalidades

- **Comandos de Moderación:** Administra tu servidor con comandos de moderación como ban, kick, mute, etc.
- **Comandos de Utilidad:** Proporciona información útil como el tiempo, información del servidor, etc.
- **Comandos de Diversión:** Ofrece comandos de entretenimiento como juegos, memes, etc.
- **Personalización:** Puedes añadir tus propios comandos y funcionalidades según tus necesidades.

## Requisitos

- Node.js
- Discord.js

## Instalación

1. Clona este repositorio:

```
git clone https://github.com/tu_usuario/bot-de-discord.git
```

2. Instala las dependencias:

```
npm install
```

3. Configura tu token de Discord en el archivo `.env`:

```
DISCORD_TOKEN=TU_TOKEN_DE_DISCORD
```

4. Ejecuta el bot:

```
node bot.js
```
